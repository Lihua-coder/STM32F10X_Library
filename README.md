# STM32F10X开源库
#### 简介
基于STM32F10X标准库V3.6.0版，逐飞科技产品及一些常用模块制作的MCU开源库。

#### 环境准备
1.  **STM32F10X硬件环境：** 
- 使用STM32F10X核心板。
2.  **软件开发环境：** 
（MDK CubeIDE IAR）
- 推荐使用 MDK。
3.  **下载方式：** 
- ST-LINK。
- UART。
- DAP。
3.  **内部晶振频率：** 
- STM32F10x 微控制器提供了适合运行的内部RC 振荡器(典型地,有8 MHz 的HSI:高速内部振荡器)。 在25 °C时,HSI 的典型精度为±1% 。 在–40 到105 °C, RC频率精度值扩大为±3% 。


#### 使用说明

1.  **下载开源库：** 点击页面右侧的克隆/下载按钮，将工程文件保存到本地。您可以使用git克隆（Clone）或下载ZIP压缩包的方式来下载。推荐使用git将工程目录克隆到本地，这样可以使用git随时与开源库保持同步。关于码云与git的使用教程可以参考以下链接 [https://gitee.com/help](https://gitee.com/help)。
2.  **打开工程：** 将下载好的工程文件夹打开（若下载的为ZIP文件，请先解压压缩包）。在打开工程前，请务必确保您的IDE满足环境准备章节的要求。
